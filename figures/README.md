# `figures/` — figures rendered for the paper

This directory holds the PNG figures that the paper embeds and that are
produced by the companion notebooks. Figures are tracked in git so that a
clean clone contains every notebook-generated output.

| File | Paper section | Producer |
|---|---|---|
| `exp_gpt2_stage6_stp_acceleration_equivalence.png` | §4 Fig. 1 | [`../notebooks/stp_loss/energy_landscape_validation.ipynb`](../notebooks/stp_loss/energy_landscape_validation.ipynb) |
| `exp_gpt2_stage6_tangential_acceleration.png`       | §5 Fig. 2 | [`../notebooks/stp_loss/energy_landscape_validation.ipynb`](../notebooks/stp_loss/energy_landscape_validation.ipynb) |
| `exp_gpt2_stage6_trajectory_acceleration_profiles.png` | §5 Fig. 3 | [`../notebooks/stp_loss/energy_landscape_validation.ipynb`](../notebooks/stp_loss/energy_landscape_validation.ipynb) |
| `sharedV_three_way.png` | §8 Fig. 4 | [`../notebooks/conservative_arch/plot_three_way_comparison.py`](../notebooks/conservative_arch/plot_three_way_comparison.py) |

> **Note.** The architecture diagram `vanilla_splm_architecture.png` (§8) is a
> static TikZ / hand-drawn figure embedded in the paper's LaTeX source; it is
> not notebook-generated and is not tracked here.

The §6 Pythia-160M replication (cross-architecture results table) and the §7
Jacobian-symmetry, §8 per-layer R² profile, §9 oracle-fit, capacity-sweep, and
token-direction figures are generated by their respective scripts but are
committed under `results/` or emitted to the notebook's working directory.
Re-run the full pipeline to regenerate:

```bash
# §4 / §5
cd ../notebooks/stp_loss/ && \
  jupyter nbconvert --to notebook --execute --inplace energy_landscape_validation.ipynb

# §6
cd ../cross_model/ && \
  jupyter nbconvert --to notebook --execute --inplace pythia_tangential_acceleration.ipynb

# §7 / §8 / §9 (assumes checkpoints are present; see checkpoints/README.md)
cd ../conservative_arch/ && \
  python jacobian_symmetry.py        && \
  python run_full_pipeline.py        && \
  python plot_three_way_comparison.py && \
  python plot_sharedV_comparison.py   && \
  python splm_oracle_fit.py           && \
  python sharedV_capacity_sweep.py    && \
  python token_direction_fit.py
```
